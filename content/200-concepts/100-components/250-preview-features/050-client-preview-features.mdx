<TopBlock>

The following Preview feature flags are available for Prisma Client:

* [`createMany`](../prisma-client/crud#create-multiple-records)
* [`orderByRelations`](../prisma-client/filtering-and-sorting#sort-by-relation-aggregate-preview)
* [`groupBy`](../prisma-client/aggregation-grouping-summarizing#group-by-preview)

To enable a Preview feature, [add the feature flag to the `client` block](./) in the `schema.prisma` file.

</TopBlock>
